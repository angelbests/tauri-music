# 基于tauri、vue、ts的播放器
本项目仅用于学习交流，不可用作其它。
## 使用
1. 本项目基于neteasecloudmusicapi
2. 安装rust环境
3. npm install
4. npm run tauri dev 运行
5. npm run tauri build 编译
